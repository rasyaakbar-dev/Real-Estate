## Creating a Property

1. Go to **Real Estate** → **Properties**
2. Click **Create**
3. Fill in property details:
   - Name, description
   - Expected price, selling price
   - Bedrooms, living area, facades
   - Garden (with area and orientation if applicable)
   - Property type and tags
4. Click **Save**

## Managing Offers

1. Open a property record
2. In the **Offers** tab, click **Add a line**
3. Enter offer price and partner (buyer)
4. Use **Accept** or **Refuse** buttons to process offers

## Property Workflow

Properties follow this lifecycle:

```
New → Offer Received → Offer Accepted → Sold
                    ↘ Canceled
```

- **New**: Initial state for new listings
- **Offer Received**: When first offer is added
- **Offer Accepted**: When an offer is accepted
- **Sold/Canceled**: Final states
