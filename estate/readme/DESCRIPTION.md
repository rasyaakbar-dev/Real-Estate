This module provides a complete real estate property management system for Odoo.

## Key Features

- **Property Tracking**: Manage properties with detailed attributes (price, bedrooms, living area, garden, availability date)
- **Offer Management**: Handle buyer offers with acceptance/refusal workflows
- **Smart Categorization**: Organize properties using hierarchical Types and flexible Tags
- **Workflow Automation**: State machine for property lifecycle (New → Offer Received → Offer Accepted → Sold/Canceled)
- **Computed Fields**: Automatic calculation of total areas and best offer prices
